About this project


Predicting Cyberbullying Comments on Social Media Using Random Forest Algorithms

Website ini adalah pengimplementasian model Machine Learning menggunakan Algoritma Random Forest untuk memprediksi komentar cyberbullying dari komentar pada social media Instagram dan Youtube.

Tujuan dari website ini adalah untuk memenuhi tugas akhir Statistika UII 2021.
